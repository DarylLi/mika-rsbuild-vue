# Vue 3 x Rsbuild

> [!NOTE]
> Use `pnpm create rsbuild@latest` instead.

## script

| 脚本           | 描述                               |
| -------------- | ---------------------------------- |
| `dev`          | 启动开发服务器                     |
| `build`        | 构建开发版本                       |
| `build-prod`   | 构建生产版本                      |
| `test`         | 启动测试环境开发服务器             |
| `preview`      | 预览构建结果                       |
| `pub-tag`      | 发布标签脚本                       |
| `format`       | 使用 Prettier 格式化所有文件       |
| `format:check` | 检查文件格式是否符合 Prettier 规范 |
| `lint`         | 使用 ESLint 检查代码质量           |
| `lint:fix`     | 自动修复 ESLint 检测到的问题       |
